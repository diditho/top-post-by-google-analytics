A WordPress plugin to display top posts in a widget using Google Analytics data via the Site Kit by Google plugin.

== Description ==

Top Post by Google Analytics is a plugin that displays the most popular posts on your WordPress site based on Google Analytics data, leveraging the Site Kit by Google plugin. This plugin provides a customizable widget that you can add to your sidebar or any widget area.

**Features:**

* Display post title and link to the top pages.
* Set the number of posts to display (default is 5).
* Requires the Site Kit by Google plugin for Google Analytics integration.

== Installation ==

1. Ensure the "Site Kit by Google" plugin is installed and activated on your WordPress site.
2. Complete the Google Analytics setup within the Site Kit by Google plugin.
3. Upload the "Top Post by Google Analytics" plugin files to the `/wp-content/plugins/top-post-by-google-analytics` directory, or install the plugin through the WordPress plugins screen directly.
4. Activate the plugin through the 'Plugins' screen in WordPress.
5. Go to "Appearance > Widgets" to add the "Top Post by Google Analytics" widget to your desired widget area.

== Frequently Asked Questions ==

= What is required for this plugin to work? =

This plugin requires the "Site Kit by Google" plugin to be installed and activated. Additionally, Google Analytics must be configured within Site Kit.

= Can I customize the number of posts displayed in the widget? =

Yes, the widget settings allow you to set the number of posts to display.

= What version of PHP is required? =

This plugin requires PHP 7.0 or higher.
