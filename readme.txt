A WordPress plugin to display top posts in a widget using Google Analytics data. Customize the widget to show post title, excerpt, number of comments, and date.

== Description ==

Top Post by Google Analytics is a simple and powerful plugin that allows you to display the most popular posts on your WordPress site based on Google Analytics data. This plugin provides a customizable widget that you can add to your sidebar or any widget area. 

**Features:**

* Display post title, excerpt, number of comments, and date.
* Set the number of posts to display (default is 5).
* Customize the length of the excerpt.
* Align the number of comments and date on the same line.
* Authenticate and retrieve data from Google Analytics with ease.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/top-post-by-google-analytics` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "Appearance > Widgets" to add the "Top Post by Google Analytics" widget to your desired widget area.
4. Go to "Settings > GA Auth Settings" to authenticate your Google Analytics account and start displaying your top posts.

== Frequently Asked Questions ==

= How do I authenticate with Google Analytics? =

After activating the plugin, go to "Settings > GA Auth Settings" and enter your Google Analytics authentication token.

= Can I customize the number of posts and excerpt length? =

Yes, the widget settings allow you to set the number of posts to display and customize the length of the excerpt.

= What version of PHP is required? =

This plugin requires PHP 8.2 or higher.
